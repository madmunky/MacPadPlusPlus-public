Help for MacPadPlusPlus

MacPadPlusPlus is a native macOS editor and large-file log viewer. Use this page for support requests, troubleshooting, and policy details.

Report a Bug Request a Feature

Common Fixes

  • Restart the app after changing language or theme settings.
  • For very large files, wait for indexing before searching.
  • If menus look stale, close and reopen the document tab.
  • Reopen recent files from File > Open Recent.

Before You Contact Support

  • App version and macOS version
  • Exact steps to reproduce
  • Expected result vs. actual result
  • Sample file or screenshot if possible
